# TalentCLEF

TalentCLEF task work for the CLEF 2026 competition.

## Folder Overview

- `Data`: Contains the training dataset and the Romanian versions of the development and test datasets given by the organizers of the competition.
- `EDA`: Contains exploratory analysis notebooks.
- `Training`: Contains model-training notebooks and translation/preprocessing experiments for the TalentCLEF task.
- `Final_model`: Contains the final solution notebook used for the selected approach.
- `backend`: Contains the API, parsing, storage, embedding, ranking, and smoke-test code for the demo application.
- `frontend`: Contains the React/Vite interface for uploading jobs/resumes and viewing rankings.

The training dataset and the Romanian versions of the development and test datasets are available here. The English and Spanish development/test sets were given by the competition.

The training dataset was extracted from Kaggle and can be found at the following link: https://www.kaggle.com/datasets/surendra365/recruitement-dataset.
